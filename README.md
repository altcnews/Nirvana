# How to Compile

```sudo apt-get -y install build-essential libssl-dev libboost-all-dev```

```sudo apt-get -y install gcc-4.8 g++-4.8  git cmake```

```git clone https://github.com/altcnews/Nirvana```

```cd Nirvana```

```mkdir build ; cd build```

```cmake ..```

```make```
